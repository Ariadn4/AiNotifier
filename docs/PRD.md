# AI Notifier - 产品需求文档

## 概述

一个轻量级 Windows 桌面悬浮球工具，用于在 AI 编程助手（Claude Code）需要用户响应时发出声音提醒。通过 Claude Code 的 Stop hook 和 Notification hook 自动触发，也可通过 HTTP 请求手动触发。

## 核心功能

### 1. 悬浮球

- 天蓝小机器人悬浮球（72×72 像素），始终置顶显示（Topmost）
- 可用鼠标拖拽移动位置
- 左键单击：在"全部关闭"和上次启用的提醒组合之间切换；提醒中点击则停止提醒回到开启
- 右键单击：弹出浅色主题菜单
- 鼠标悬停：弹性放大 1.1 倍 + Tooltip 显示状态
- 小球外观根据状态变化（机器人形象 + 渐变球体）：
  - **监听中（ON）**：天蓝渐变球（#7dd3fc → #0284c7），机器人微笑 + 偶尔眨眼 + 天线柔光
  - **已关闭（OFF）**：灰调渐变球（#6b7280 → #374151），机器人打瞌睡 zzZ + 灰调静态
  - **回应完毕提醒中（ALERT-STOP）**：琥珀渐变球（#fbbf24 → #d97706），机器人惊喜圆眼 + 摇摆 + 脉冲波纹
  - **通知提醒中（ALERT-NOTIFICATION）**：紫色渐变球（#c084fc → #9333ea），机器人惊喜圆眼 + 摇摆 + 脉冲波纹

### 2. 双类型声音提醒

支持两种独立的提醒类型，用户可分别开关：

- **通知提醒**（AI 发送通知时）：通过 `http://localhost:19836/notify` 触发，紫色球体，默认提示音3
- **回应完毕提醒**（AI 回应完毕时）：通过 `http://localhost:19836/stop` 触发，橙色球体，默认提示音1

两种提醒共享相同的行为逻辑，仅球体颜色和音效不同。

#### 提醒开关

通过右键菜单"开启提醒"子菜单独立控制：
- AI 发送通知时提醒（勾选/取消）
- AI 回应完毕时提醒（勾选/取消）
- 全部关闭

左键点击悬浮球在"全部关闭"和"用户上次启用的组合"之间切换。

#### 提醒行为

- 支持两种提醒模式（通过右键菜单切换）：
  - **长提醒**（默认）：循环播放提醒音，停止条件（满足任一即停）：
    - 用户点击悬浮球
    - 用户有键盘/鼠标活动（检测到用户回来了）
    - 响铃超过设定时长自动停止（可配置 15/30/45/60 秒，默认 60 秒）
  - **短提醒**：播放一次提示音，约 4 秒后声音自动停止
- 无论长提醒还是短提醒，悬浮球的外观变化（闪烁/摇摆/波纹）会持续保持，直到检测到用户有键盘/鼠标活动后才恢复为"监听中"样式
- 无论长提醒还是短提醒，无论是否被打断，至少播放完整 1 次音效后才停止声音
- **渐强音量**（可选）：仅长提醒模式下有效，开启后提醒声音从 1% 音量开始，以 30 秒升到 100% 的固定速率逐渐增大到设定音量
- 提醒功能关闭时，收到信号不响铃

### 3. 音效设置弹窗

通过右键菜单"提示音效"打开设置弹窗，集中管理：
- **回应完毕提示音**：4 个内置音效 + 自定义文件选择，hover 预览
- **通知提示音**：4 个内置音效 + 自定义文件选择，hover 预览
- **音量**滑块（0-100%）
- **渐强音量**开关

### 4. 碎碎念

- 通过 HTTP 请求触发气泡（`http://localhost:19836/bubble`）
- 悬浮球旁弹出小气泡，随机显示一条提醒消息
- 默认消息："喝杯水吧"、"站起来走走"、"眺望远处20秒"
- 用户可通过右键菜单 → 碎碎念 → 编辑碎碎念内容 自定义消息内容
- 气泡动画：淡入 → 停留 5 秒 → 淡出
- 可配置冷却时间（无冷却 / 5 / 10 / 15 / 30 分钟），防止频繁弹出
- 默认无冷却（每次请求都弹）
- 可通过右键菜单开关碎碎念功能

### 4.1 项目通知气泡

- 当 Stop 或 Notification 请求携带 `cwd` 参数时，在悬浮球旁显示项目通知气泡
- 显示项目名（取工作目录最后一级）+ 事件类型，如 "AiNotifier 回复完毕"、"MyProject 发送通知"
- 多个通知会累积显示在同一个气泡中（每行一条），方便用户离开后回来查看
- 点击气泡立即关闭；键盘/鼠标有活动后 5 秒自动消失
- 可通过右键菜单 → 碎碎念 → 项目通知气泡 开关
- 开启时 Claude Code hook 命令自动携带 `$CLAUDE_PROJECT_DIR` 环境变量

### 5. 开机自启动

- 右键菜单中可切换"开机自启动"
- 通过写入注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 实现
- 默认不开启

## 与 Claude Code 的集成

绑定 Claude Code 时，同时添加三个 hook：

```json
{
  "hooks": {
    "Stop": [
      {
        "hooks": [{ "type": "command", "command": "curl -s \"http://localhost:19836/stop?cwd=$CLAUDE_PROJECT_DIR\"" }]
      }
    ],
    "Notification": [
      {
        "hooks": [{ "type": "command", "command": "curl -s \"http://localhost:19836/notify?cwd=$CLAUDE_PROJECT_DIR\"" }]
      }
    ],
    "UserPromptSubmit": [
      {
        "hooks": [{ "type": "command", "command": "curl -s http://localhost:19836/bubble" }]
      }
    ]
  }
}
```

也可通过手动执行 HTTP 请求触发：
```bash
curl http://localhost:19836/stop           # 回应完毕提醒
curl http://localhost:19836/notify         # 通知提醒
curl http://localhost:19836/bubble          # 气泡提醒
```

## 技术选型

- **语言**：C#（WPF）
- **框架**：.NET 8（独立部署，无需安装运行时）
- **音频**：System.Media.SoundPlayer
- **HTTP 服务**：内置 HttpListener 监听本地端口
- **用户活动检测**：Win32 API `GetLastInputInfo`

## 界面设计

详细设计方案参见 `docs/ai-ping-skyblue-robot.html`（天蓝小机器人设计稿）。

- 悬浮球 72×72 像素，天蓝小机器人形象
- 四种状态：天蓝渐变（监听中）/ 灰调（已关闭）/ 琥珀渐变（回应完毕提醒中）/ 紫色渐变（通知提醒中）
- 机器人表情随状态变化：微笑眨眼 / 打瞌睡 zzZ / 惊喜圆眼（两种提醒共用表情，颜色不同）
- 浅色主题右键菜单（白色背景，圆角 12px）
- 应用图标：深蓝夜色背景 + 白色机器人 + "AI PING" 文字，多尺寸（256/64/48/32/16）

## 非功能需求

- 内存占用 < 30MB
- CPU 空闲时趋近 0%
- 单实例运行（不允许重复打开）
